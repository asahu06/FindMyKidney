FindMyKidney is a Streamlit-powered tool designed to optimize kidney transplant allocation. It modernizes the matching process by calculating KAS, KDPI, and EPTS scores to reduce wait times and kidney discard rates.

Key Features:
  Patient Registry: Track and manage waitlisted candidates.
  KidneyMatch Engine: Real-time matching based on donor characteristics.
  Dual-Track Allocation: Parallel processing for marginal kidneys (KDPI > 20%) to reduce refusal cycles.
  Smart Scoring: Automated calculation of clinical priority and acceptance likelihood.

Tech Stack
  Language: Python
  Framework: Streamlit
  Data Handling: Pandas
